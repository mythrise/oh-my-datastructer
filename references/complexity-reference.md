# 时空复杂度速查表

供 `/ds:report` 和 `/ds:get-information` 引用的标准复杂度参考。

---

## 排序算法

| 算法 | 最好 | 平均 | 最坏 | 空间 | 稳定 |
|------|------|------|------|------|------|
| 冒泡排序 | O(n) | O(n²) | O(n²) | O(1) | ✓ |
| 选择排序 | O(n²) | O(n²) | O(n²) | O(1) | ✗ |
| 插入排序 | O(n) | O(n²) | O(n²) | O(1) | ✓ |
| 希尔排序 | O(n log n) | O(n^1.3) | O(n²) | O(1) | ✗ |
| 快速排序 | O(n log n) | O(n log n) | O(n²) | O(log n) | ✗ |
| 归并排序 | O(n log n) | O(n log n) | O(n log n) | O(n) | ✓ |
| 堆排序 | O(n log n) | O(n log n) | O(n log n) | O(1) | ✗ |
| 计数排序 | O(n+k) | O(n+k) | O(n+k) | O(k) | ✓ |
| 桶排序 | O(n+k) | O(n+k) | O(n²) | O(n+k) | ✓ |
| 基数排序 | O(d·(n+k)) | O(d·(n+k)) | O(d·(n+k)) | O(n+k) | ✓ |
| TimSort | O(n) | O(n log n) | O(n log n) | O(n) | ✓ |

## 图算法

| 算法 | 时间复杂度 | 空间复杂度 | 适用 |
|------|-----------|-----------|------|
| BFS | O(V+E) | O(V) | 无权最短路 |
| DFS | O(V+E) | O(V) | 连通性、拓扑排序 |
| Dijkstra (数组) | O(V²) | O(V) | 稠密图单源最短路 |
| Dijkstra (堆) | O((V+E)log V) | O(V) | 稀疏图单源最短路 |
| Bellman-Ford | O(VE) | O(V) | 含负权边 |
| Floyd-Warshall | O(V³) | O(V²) | 全源最短路 |
| Prim (数组) | O(V²) | O(V) | 稠密图 MST |
| Prim (堆) | O(E log V) | O(V) | 稀疏图 MST |
| Kruskal | O(E log E) | O(V) | 稀疏图 MST |
| A* | O(E) ~ O(b^d) | O(b^d) | 启发式搜索 |

## 哈希表

| 方法 | 查找平均 | 查找最坏 | 插入平均 | 空间 |
|------|---------|---------|---------|------|
| 链地址法 | O(1+α) | O(n) | O(1) | O(n+m) |
| 开放寻址 | O(1/(1-α)) | O(n) | O(1/(1-α)) | O(m) |
| Cuckoo 哈希 | O(1) | O(1) | 摊还 O(1) | O(n) |
| 完美哈希 | O(1) | O(1) | O(n) 构建 | O(n) |

> α = n/m（负载因子），n = 元素数，m = 桶数

## 字符串匹配

| 算法 | 预处理 | 匹配 | 空间 |
|------|--------|------|------|
| 暴力匹配 | O(1) | O(mn) | O(1) |
| KMP | O(m) | O(n) | O(m) |
| Boyer-Moore | O(m+σ) | O(n/m) ~ O(mn) | O(m+σ) |
| Rabin-Karp | O(m) | O(n) 期望 | O(1) |
| AC 自动机 | O(Σm_i) | O(n+k) | O(Σm_i·σ) |

> n = 文本长度，m = 模式长度，σ = 字符集大小，k = 匹配数

## 编码与压缩

| 算法 | 编码时间 | 解码时间 | 压缩率 |
|------|---------|---------|--------|
| 哈夫曼编码 | O(n log n) | O(n) | 接近熵 |
| 规范哈夫曼 | O(n log n) | O(n) | 同上，头部更小 |
| LZW | O(n) | O(n) | 适合重复模式 |
| 算术编码 | O(n) | O(n) | 最接近熵 |
